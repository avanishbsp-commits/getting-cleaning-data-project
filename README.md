getting-cleaning-data-project
Getting and Cleaning Data Course Project

This project demonstrates data cleaning using the Human Activity Recognition dataset.

Steps performed:
1. Merged training and test datasets
2. Extracted mean and standard deviation features
3. Applied descriptive activity names
4. Renamed variables clearly
5. Created a tidy dataset with averages for each activity and subject

Run run_analysis.R to generate tidy_data.txt.
